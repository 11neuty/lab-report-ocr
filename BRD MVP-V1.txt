# 1. EXECUTIVE SUMMARY

Saat ini hasil pengujian laboratorium udang dicatat menggunakan formulir fisik yang ditulis tangan oleh analis laboratorium.

Data hasil pengujian kemudian harus dimasukkan kembali secara manual ke dalam file Excel atau CSV untuk kebutuhan pelaporan dan analisis.

Proses ini menimbulkan beberapa permasalahan:

* Membutuhkan waktu yang lama.
* Berisiko terjadi kesalahan input.
* Duplikasi pekerjaan.
* Sulit melakukan pencarian data historis.

Aplikasi OCR yang diusulkan akan memungkinkan pengguna mengunggah hasil scan formulir laboratorium, melakukan ekstraksi data secara otomatis menggunakan OCR, memberikan kesempatan kepada pengguna untuk melakukan koreksi apabila terdapat kesalahan pembacaan, kemudian menghasilkan file CSV yang siap digunakan.

# 2. BUSINESS OBJECTIVES

## Objective 1

Mengurangi proses input data manual dari formulir laboratorium.

Target:

* Pengurangan waktu input minimal 70%.

## Objective 2

Mengurangi human error saat entry data.

Target:

* Akurasi data akhir minimal 95%.

## Objective 3

Mempercepat proses digitalisasi hasil laboratorium.

Target:

* 1 dokumen dapat diproses kurang dari 1 menit.

## Objective 4

Menghasilkan file CSV secara otomatis.

Target:

* Tidak diperlukan re-entry ke Excel.
