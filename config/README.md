# Configuration

This directory contains shared configuration files for lab-report-ocr.

Future files will include:

- `eslint.config.js` — ESLint configuration for frontend.
- `prettier.config.js` — Prettier formatting configuration.
- `tsconfig.base.json` — Base TypeScript configuration.
- `pyproject.toml` — Python project and tool configuration (Ruff, pytest).
- `vitest.config.ts` — Base Vitest configuration.
